### 📖 简介：

WhatsApp盗号系统/快产码WS盗号/网页版盗Ws系统/Java全开源源码是java全开源的盗WhatsApp系统，快产码

---

### 💡 系统亮点：

- **混合架构设计**：B/S (管理后台) + C/S (自动化客户端) 架构，实现 web 端远程控制本地浏览器集群
- **实时通信**：WebSocket 毫秒级同步状态，确保账号状态实时更新
- **多开隔离**：自动管理 Chrome User Data Dir，实现浏览器指纹隔离
- **跨设备登录**：通过 Session 备份技术，实现跨设备/跨进程免密登录
- **智能代理系统**：支持直连、全局代理、IP 代理池（BlurPath 集成）
- **元数据自动采集**：自动解析并存储账号的操作系统、IP 代理、归属地等信息
- **IP 归属地显示**：实时调用 ip-api.com 接口，直观显示 IP 地理位置

---

### 💻 运行环境及框架：

**后端 / 管理后台 ( wsapp-admin-master-2.0 )**

- 基础框架：RuoYi (若依) v4.7.7
- 开发语言：Java 8 (JDK 1.8)
- 核心组件：
  - Spring Boot 2.5.14
  - Apache Shiro 1.10.1
  - MyBatis
  - Thymeleaf
  - WebSocket
  - Fastjson 1.2.83
- 数据库：MySQL 5.7 / 8.0
- 运行容器：内置 Tomcat

**自动化客户端 ( wsapp-client-pro )**

- 开发语言：Java 8 (JDK 1.8)
- GUI 框架：Java Swing
- 自动化引擎：Selenium 4.10.0
- 通信协议：
  - WebSocket (Java-WebSocket 1.5.3)
  - HTTP (Apache HttpClient 4.5.3)
- 关键依赖：Lombok, Jsoup, Commons-IO

**开发环境**

- JDK：Java 1.8
- Maven：3.x
- IDE：IntelliJ IDEA
- 浏览器：Google Chrome (需配合对应版本的 chromedriver.exe)
